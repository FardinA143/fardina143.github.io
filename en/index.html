Raspberry Pi

A Raspberry Pi is a small device of the size of a credit card, on which you can develop various projects, such as a small PC, a media center, task automation device, a Raspberry Pi cluster (More information at MagPi), and others. It can also be deployed in the world of Internet of Things.
In my case, I previously used my Raspberry Pi for the development of various things, such as Python scripts, and also this website (until I got a new PC)

Python

Python is a high level programming language. It can be used in various platforms, such as Windows, MacOS, Linux-based OS, and also in mobile devices. This language is also used in development of Progressive Web Apps (PWA for short).

GitHub
Code version control

GitHub is an online service for code version control developed by GitHub Inc. (although Microsoft bought it some months ago). It's used mainly to save and share code of big projects. Each project is saved in repositories. Recently they also provided a method to create a webpage for each repository. You can access my GitHub with the link on the footer.
