Raspberry Pi

A Raspberry Pi is a small device of the size of a credit card, on which many projects can be developed, such as a small PC, a media center, task automation device, a Raspberry Pi cluster-computer, and other things. It can also be deployed for the world of Internet of Things.
In my case, I previously used my Raspberry Pi for the development of various things, such as Python scripts, server hosting. This webpage was also hosted on a Raspberry Pi, but because of it's reduced size, and thus reduced computing power, I had to move it on another server.

Python

Python is a high level programming language. It can be used in various platforms, such as Windows, MacOS, Linux-based OSs, and also in mobile devices. This language is also used in development of Progressive Web Apps (PWA for short). This language is very popular because of its ease of use and ease of learn. It's also known because of it's wide application scenarios.
