IoT
Raspberry Pi

Una Raspberry Pi es un dispositivo del tamaño de una tarjeta de crédito, con la cual se desarrollan varios proyectos, tales como un mini-PC, un HTPC, automatización de tareas, e incluso un superordenador (más información en MagPi.) Es uno de los dispositivos que se puede usar como dispositivo de IoT (Internet de las cosas, del inglés Internet of Things ).
En mi caso, el uso que le doy a una Raspberry Pi para el desarollo de varias cosas (entre ellos scripts de Python y también esta misma página web.)

Python

Python es un lenguaje de programación multiuso. Se puede usar en varias plataformas, ya sea en sistemas operativos como puede ser Windows, MacOs, sistemas Linux, en dispositivos móviles, e incluso en desarollo de páginas web. Python es un lenguaje fácil de aprender, pero no obstante, es un lenguaje bastante potente. Python es uno de los lenguajes de programación que más uso para el desarollo de scripts.

GitHub
Control de código

GitHub es un servicio en la nube, desarollado por GitHub Inc., (actualmente subsidiaria de Microsoft) que sirve para proporcionar acceso al código fuente del programario creado de manera pública. Muchos scripts mios se encuentran en GitHub. Cada proyecto se nombra repositorio, y es posible disponer una página de introducción para ese proyecto. Puedes obtener acceso a mi perfil de GitHub en el pie de esta página.
